返回目录表名：SALES
说明：销售表
数据列：
序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 SALES_ID VARCHAR(30) 30 0 N Y 销售ID
2 DISH_ID VARCHAR(20) 20 0 N 菜品ID
3 DISH_PRICE float 5 2 N 下单时菜价
4 ORDER_ID VARCHAR(25) 25 0 N 订单编号（外键）
5 SAL_STATUS NUMBER(1) 1 0 N 状态，0已删除 >=1有效 1已下单 2正在做 3做完了
返回目录表名：RES_TABLE
说明：餐桌表
数据列：
序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 TABLE_ID VARCHAR(20) 20 0 N Y 餐桌ID
2 TABLE_NUMBER VARCHAR(20) 20 0 N 餐桌编号
3 DEFAULT_NUMBER NUMBER(3) 3 0 N 规定人数
4 TABLE_ORDER_STATUS NUMBER(1) 1 0 N 是否有人，0为无人，1为已预订，2为已上座
5 TAB_STATUS NUMBER(1) 1 0 N 状态，0已删除 1有效
返回目录表名：PRE_ORDER
说明：预定表
数据列：
序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 PREORDER_ID VARCHAR(25) 25 0 N Y 预订ID
2 PREORDER_TIME VARCHAR(20) 20 0 N 预订下单时间
3 ARRIVE_TIME VARCHAR(20) 20 0 N 预约上座时间
4 ORDER_ID VARCHAR(25) 25 0 N 订单ID（外键）
5 DISH_LIST VARCHAR(1000) 1000 0 Y 点菜列表，dish_id用逗号分隔
6 TABLE_ID VARCHAR(20) 20 0 N
7 PRE_STATUS NUMBER(1) 1 0 N 预订状态，0已删除 >1有效 1为已预订还没到 2为已上座 3为已取消预订
返回目录表名：PRESENCE
说明：出勤表

序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 PRESENCE_ID VARCHAR(25) 25 0 N Y 出勤ID
2 SIGN_TIME VARCHAR(25) 25 0 N 签到时间
3 PRE_MONTH NUMBER(2) 2 0 N 月份
4 EMPLOYEE_ID VARCHAR(20) 20 0 N 员工ID（外键）
5 HASPRESENTED NUMBER(1) 1 0 N 是否出勤，0为未出勤，1为已出勤
6 PRE_STATUS NUMBER(1) 1 0 N 状态，0为已删除，1为有效
返回目录表名：OVERHEAD
说明：开销表
数据列：
序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明

2 OVERHEAD_TYPE NUMBER(1) 1 0 N 开销类型，1为进货，2为水电费，3为房租，4为其他
3 OVERHEAD_PRICE float 10 2 N 总金额
4 OVERHEAD_DATE VARCHAR(10) 10 0 N 开销日期

返回目录表名：ORDER_LIST
说明：订单表
数据列：
序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 ORDER_ID VARCHAR(25) 25 0 N Y 订单ID
2 TABLE_ID VARCHAR(20) 20 0 N 餐桌ID（外键）
3 DISH_LIST VARCHAR(1000) 1000 0 Y 点菜列表，dish_id用逗号分隔
4 TOTAL_PRICE float 6 2 Y 总价

7 PAY_TIME VARCHAR(20) 20 0 Y 结账时间
8 ORDER_NOTE VARCHAR(120) 120 0 Y 备注
9 ORD_STATUS NUMBER(1) 1 0 N 订单状态，0已删除 1有效
